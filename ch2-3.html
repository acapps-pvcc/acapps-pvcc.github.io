Chapter 2: Briefly define:
  1. natural language: It is the language we speak and write in our everyday lives. This includes language such as English, Spanish, Arabic, Japanese, Swahili, or any language.

  2. pseudocode: A set of English-language constructs that resemble programming statements but don't run on a computer. It represents a compromise between the two extremes of natural and formal languages.

  3. sequential operation (straight-line algorithm): A type of algorithm that executes its instructions in a straight line from top to bottom and then stops, It doesn’t include decision making, branching, and repetition.

  4. input: Operations provide the computing agent with data values from the outside world that it may then use in later instructions.

  5. output: Operations send results from the computing agent to the outside world.

  6. conditional statement: Are the “question-asking” operations of an algorithm. They allow an algorithm to ask a yes/no question and select the next operation to perform on the basis of the answer to that question.

  7. boolean condition: A logical expression that evaluates to true or false and determines which condition of an if/then/else statement to execute.

  8. algorithmic discovery process: This involves finding a solution to a given problem. It is the most challenging and creative part of the problem-solving process.
Chapter 3: Briefly describe or define:
  1. why algorithms should be correct: Beacuse an algorithm that gives wrong results for the problem at hand is worse than useless. It can lead to mistakes that are enormously expensive or even fatal.

  2. why algorithms should be efficient: Algorithms should be efficient because as computer memory capacity, processing speed, and problem complexity increase, resources are limited. Some algorithms consume so many resources that they won't be practical.

  3. program maintenance: After a program is written it needs to be modified; correct errors that are uncovered through repeated usage with different input values and to extend the program to meet new requirements or to expand its functionality.

  4. importance of algorithm: These are to make an algorithm easy to understand, correct, and clear. Regarding correct and accurate logic, it should be structured in a way that is as easy to follow as possible, which is a highly desirable characteristic of an algorithm.

  5. algorithmic elegance: This is defined as the algorithmic equivalent of style. It refers to an algorithm that is fairly easy to read, and provides the same handling, and a clever solution to a problem.

  6. sequential search algorithm: It is the comparison of the NUMBER being searched for against a single number in the list.

  7. order of magnitude: Is used to analyze the time efficiency of the sequential search algorithm. It refers to how an algorithm's performance such as time and memory usage grows as the input size n increases.

  8. selection sort: An algorithm that repeatedly searches for the smallest value in the unsorted section of the list, and swaps it with the first element of the section. It continues the process, moving the vertical line between the sorted section at the front of the list and the unsorted section at the back, until the entire list is sorted.

  9. flop (floating point operation): Ways to compare performance among different makes of computers is to give the number of arithmetic operations, such as additions or subtractions of real numbers, that each one can do in 1 second.

  10. teraflop: This refers to 16 trillion floating-point operations per second; that’s 16,000,000,000,000.

  11. petaflop: This refers to 93 quadrillion floating-point operations per second; that’s 93,000,000,000,000,000

  12. parallel processor: It is a world’s top-speed supercomputer that was declared by Sunway TaihuLight at the National Supercomputing Center in Wuxi, China, and it has 10,649,600 core processors.

  13. data cleanup problem: Is the process of identifying and removing unwanted elements, such as zeros, from an original list to ensure the values are legitimate and accurate.

  14. binary search algorithm: This is an efficient algorithm used to find the position of a target value in a sorted data list. If the number there equals NUMBER, the search is over. If NUMBER comes numerically before the number at the halfway point, then the search is narrowed to the front half of the list, and the process begins again on this smaller list. If NUMBER comes numerically after the number at the halfway point, then the search is narrowed to the back half of the list. The process repeatedly divides the list in half until the sublist becomes empty.

  15. pattern matching: It refers to the process of searching for a special pattern of symbols within a larger collection of information.
Karl Friedrich Gauss
  1. Give a brief overview of Karl Friedrich Gauss' mathematics abilities and scientific achievements. Karl Friedrich Gauss's mathematical abilities include fundamental results in number theory, which he introduction of modular arithmetic, a system in which numbers “wrap around” after reaching a certain value. Algebra, with its fundamental theorem of algebra. It states that every polynomial equation with real or complex coefficients has as many roots. In addition, He made numerous contributions, such as the composition law, proved the law of quadratic reciprocity, and the triangular case of the Fermat polygonal number theorem. In terms of Karl Friedrich Gauss's scientific achievements include he invented an early type of magnetometer, a device that measures the direction and strength of a magnetic field. His practical work also led to the invention of the heliotrope in 1821. Gauss also developed a consistent system of magnetic units, and with Wilhelm Weber built one of the first electromagnetic telegraphs in 1833.

  2. Describe Karl Frederich Gauss' algorithm for adding up the numbers from 1 to 100: Karl Frederich Gauss’s algorithm for adding 1 to 100 is the sum of the first number and the last number, 1 and 100, which is 101. And so was the sum of the second and second-to-last numbers, 2 and 99. Gauss realized that this applied to all 100 numbers. And since 50 such pairs existed, all he needed to do was multiply 101 by 50 to get 5050, the correct answer.

  3. What are gaussian optics? Gaussian optics is a technique in geometrical optics that describes the behaviour of light rays in optical systems by using the paraxial approximation, in which only rays which make small angles with the optical axis of the system are considered.

  4. What is a "gaussian blur" and how is it used in digital image editing? Gaussian Blur is a filtering technique that uses a Gaussian function to create a smoothing effect on an image. The blur reduces image noise and reduces detail by averaging pixel values with their neighbors.
